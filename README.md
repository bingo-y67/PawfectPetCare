## Description

Pawfect Pet Care is a single-page Bootstrap 5 site for a fictional pet grooming, daycare, and boarding business.  
The page includes:

- A sticky Bootstrap navbar with smooth scrolling
- A hero section introducing the business
- A pet gallery using the Bootstrap carousel component
- A services section with three responsive cards (daycare, grooming, boarding)
- A FAQ section built with the Bootstrap accordion component
- Fully responsive layout for mobile, tablet, and desktop
